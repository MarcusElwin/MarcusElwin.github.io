## Typo checks
This repo uses [typos](https://github.com/crate-ci/typos/tree/master) for typo checks to run locally:
```sh
# check typos in content folder
typos ./ds-with-mac/content/ 
```

```sh
# correct typos in folder
typos ./ds-with-mac/content/ -w
```
